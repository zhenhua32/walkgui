# walkgui

## 简介
这是一个爬虫, 用来下载图片. 具体分为三步
* 下载首页
 * 首页是包含列表的页面, 通常来说具有很多导航列表
* 下载具体页面
 * 具体页面是包含图片的页面, 由首页跳转而来
* 下载图片
 * 图片就是要获取的资源
 
需要正则表达式, 技能自备.

因为获取链接是依赖正则表达式, 所以正则的好坏
很大程度上影响图片的获得.

## 依赖
需要nodejs>=4.x和npm

## 安装
使用 `npm start` 安装必要依赖

## 启动
使用 `npm start` 运行

## 构建
使用 `npm run build` 构建

可以修改 build.js 文件来修改生成的程序文件夹,
具体请参考 electron-packager,
默认只生成 win32 下的程序.
直接用 npm start 也是可以了, 虽然要从命令行里启动.
